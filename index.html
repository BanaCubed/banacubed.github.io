My Projects

Theta Incremental

Theta Incremental is currently on hiatus.

Currently has one prestige layer: Ranks.

Theta Incremental is my first every serious coding project, and has been remade twice. Originally made in Twine, the game had some serious fundamental issues caused by both a lack of knowledge and limitations of Twine itself. Near the end of 2024, I remade the game using The Modding Tree.

Create Incremental

Create Incremental is currently being worked on.

Currently has a total of three prestige layers: Rebirth, Super Rebirth, Hyper Rebirth.

Create Incremental has features primarily originating from a forum thread on galaxy. The currently worked on remake of the game (third fourth time's the charm!) puts added focus on "The Machine" as a mechanic
